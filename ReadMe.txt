Steps to compile and run the project




1. Login to university server and go to public_html 

2. Unzip the file on desktop and transfer into public_html folder
3. We have the below files in the project.
	
	
   a. AlphaBeta.java
	
   b. Board.java	
   c. Console_Interface.java
   d. Evaluations.java
   e. MiniMax.java 
4. First compile the prohram using following command: (to create obj file)

	javac Console_Interface.java

5. Then run the program using following command:

	java Console_Interface
